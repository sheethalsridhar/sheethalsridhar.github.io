Control of Cyberbullying Using Semantic-Enhanced Marginalized Denoising Auto-Encoder

Objective: “ To implement a safe and secure online social network that is free from textual based cyberbullying. Not just when there are direct bullying words but also when there are hidden features in it and thereby blocking such posts. On repetitive bullying the user is blocked permanently. ”

Methodology: Developed a web application/Online Social Networking (OSN) with the basic features where new users can register, existing users can login, send messages privately and publicly, share posts, accept and send friend requests.

Constructed the bullying feature set by adding bullying words to the existing corpus using word embedding. Assigned weights to each of these words. Found the intersection between the Bag of Words (BoW) and bullying dictionary and appended it to the bullying features.

Used the learning method Semantic-Enhanced Marginalized Denoising Auto-Encoder (smSDA) to exploit the hidden feature structure of textual based bullying information.Trained the smSDA, to reconstruct bullying features from other normal words by discovering the correlation between bullying and normal words to detect the bullying messages.

Notified the user account if a bullying word is found and blocked the user if the bullying is repetitive.

  • Tools Used: MySQL Database, Apache Server, Java, JDBC, PHP, NetBeans IDE
  • Code: View Code and Insights
  • Date: Jan 2018-May 2018
  • Category: Machine Learning & Data Science
